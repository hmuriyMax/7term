Основные понятия

Начнем с общей постановки линейной многокритериальной задачи.

Задача.

Предположим заданными:

  • область w изменения допустимых значений переменных х1,..., хn, определяемую совокупностью линейных уравнений и неравенств, и
  • набор критериев Cr1, ..., Crm, оценивающих качество искомого решения.

Будем считать, что каждый из этих критериев линейно связан с переменными х1, ..., хn,

где γik — известные числа.

В области w требуется найти такой набор переменных (х1, ..., хn), при котором по всем критериям достигались бы максимальные значения,

Cr1 → max, …, Crm → max.

Метод свертывания описывается следующим образом. Лицо, принимающее решения, из некоторых, часто доступных только ему соображений назначает веса критериев,

w1, ..., wi, ..., wm,

w1≥ 0, ..., wi≥ 0, ..., wm≥ 0,

что позволяет свернуть заданные критерии в один глобальный критерий,

Crgl=w1Cr1+...+wiCri+...+wmCrm,

и свести исходную задачу к обычной задаче линейного программирования с одним критерием: найти в области w такой набор переменных (х1, ..., хn), при котором глобальный критерий достигает максимального значения,

Crgl → max.
